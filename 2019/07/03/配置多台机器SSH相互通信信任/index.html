配置多台机器SSH相互通信信任

2019-07-03
Linux

本文详细介绍如何配置多台机器SSH互通，以及可能遇到的坑。

5台机器执行 ssh-keygen

选取第一台,生成authorized_keys文件

然后将其他四台的id_rsa.pub内容,手动copy到第一台的authorized_keys文件

权限(每台机器)

将第一台的authorized_keys scp 给其他四台(第一次传输,需要输入密码)

验证(每台机器上执行下面5条命令,只输入yes,不输入密码,则这5台互相通信了)
